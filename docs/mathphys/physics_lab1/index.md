## 3 考核方式
- 实验70分（即每次实验5分）。
- 期末闭卷理论考试30分，考察绪论、两个必做实验。~~考前预习要求~~
## 4 课程内容
第一节课为绪论课，主要讲了不确定度等实验数据处理的内容，从第二节课开始做实验。

- **示波器的应用（必做实验）**

- **弗兰克赫兹实验,交流电路功率因数实验 (同时做两个实验内容)**
- **用霍尔法测直流圆线圈与亥姆霍兹线圈磁场**
